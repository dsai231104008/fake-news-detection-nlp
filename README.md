📰 Fake News Detection System Using NLP and ML
📌 Project Overview :

This project implements a Fake News Detection System using Natural Language Processing (NLP) and Machine Learning techniques.
The system analyzes the textual content of news articles and predicts whether the news is Fake or Real.

The trained model is integrated into a local graphical user interface (GUI) developed using Streamlit, and the application is executed locally through VS Code.

🎯Project Objectives:
Detect fake and misleading news articles
Apply NLP techniques for text preprocessing
Convert text data into numerical features
Train a machine learning classification model
Develop a simple and interactive local GUI

Natural Language Processing (NLP)
